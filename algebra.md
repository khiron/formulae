# Algebra 

## Logarithm rules

- ### $\log m n = \log m + \log n$
- ### $\log \frac{m}{n} = \log m - \log n$
- ### $\log m^n = n \cdot \log m$
- ### $\log_n m = \frac{\log m}{\log n}$
- ### $\log_x x = 1$
- ### $e^{log x} = x$
- ### $\log e^x = x$
- ### $\log_a 1 = 0$
- ### $\log_a a = 1$
- ### $a^{\log_a n} = n$
- ### $\log_a b = \frac{1}{\log_b a}$
